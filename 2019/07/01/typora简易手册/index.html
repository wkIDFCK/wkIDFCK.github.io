typora简易手册

撰写于 2019-07-01 修改于 2019-07-01 标签 typora建议手册，时时看

区域元素

1.分割线

​ 输入 *** / —，换行键换行

2.段落

​ 按换行键

3.标题

​ 开头#的个数（谨记加上文字之间空格），标题有1~6个级别，#表示开始，按换行键结束

4.引注

​ 开头 “>” + 空格 + 文字 ，按换行键换行，双按换行跳出

wang
zhe
rong
yao

5.序列

​ 开头 “ * / + / - ” + 空格 + 文字，可以创建无序序列，换行键换行，删除键+shift+tab跳出

  • wang
  • zhe
  • rong

​ 开头 “ 1. ” + 空格 + 文字，可以创建有序序列

  1. wang
  2. zhe

6.可选序列(创建选项)

​ 开头 “ * / + / - ” + 空格 + [ ] + 空格 + 文字，换行键换行，删除键+shift+tab跳出

  • a
  • b
  • c
  • completed

7.代码块

​ 开头 “

8.数学块

​ 开头 ” $$ “ + 换行键，产生输入区域，输入Tex/LaTex格式的数学公式

$$
\mathbf{V}_1 \times \mathbf{V}_2 = \begin{vmatrix}
\mathbf{i} & \mathbf{j} & \mathbf{k}
\frac{\partial X}{\partial u} & \frac{\partial Y}{\partial u} & 0
\frac{\partial X}{\partial v} & \frac{\partial Y}{\partial v} & 0
\end{vmatrix}
$$

9.表格

开头 “ | ” + 列名 + “ | ” + 列名 + “ | ” + 换行键，创建一个2*2表格

将鼠标放置其上，弹出编辑尺寸，个数，文字等

1 2

10.脚注

在需要添加脚注的文字后面 + [ + ^ + 序列 + ]，注释的产生可以鼠标放置其上单击自动产生，添加信息

或人工添加 + [ + ^ + 序列 + ] + :

脚注产生的方法^footnote.
