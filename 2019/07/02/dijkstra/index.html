dijkstra算法的一般原理及实现

​ 初始变量：VertexsNum //顶点个数

​ Arc[VertexsNum] [VertexsNum] //图的存储结构

​ S[VertexsNum] = {0}; //已考察顶点的标记，初始为0；若已考察，置1

​ dist[VertexsNum] //到该顶点当前的最小距离,dist[] 的初始值dist[i] = arcs[0][i]；

​ path[VertexsNum] //表示到该顶点当前最短路径的前驱节点

1. 原理

step 1:

​ 将源点的S[]数组对应位置置1，dist[] 的初始值dist[i] = arcs[0][i]；

step2:

​ 从顶点集合V-S中选出Vj，该Vj满足dist[j] = Min{dist [i] | Vi ∈ V-S} ，将Vj并入S；

step3：

​ 修改从V0出发到集合V-S上其他顶点Vk可达的最短路径长度：如果dist[j] + arcs[j][k] < dist[k]，则令dist[k] = dist[j] + arcs[j][k]；

step4：

​ 重复 2~3，共操作n - 1次，直到所有顶点都包含在S中。
